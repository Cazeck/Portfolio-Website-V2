The Warehouse

Amazon’s automated fulfillment centers are incredibly complex hubs of activity where humans and robots work in unison.

This collaboration of people, machine learning, and computer algorithms result in incredible efficiencies in Amazon's daily operations. These efficiences allow for Amazon to ship over 1 million packages per day.

The Simulation

This project was an assignment presented in my Object-Oriented Design class that I thought to be very interesting. However, we didn't go into it in enough depth so I thought this would be an interesting opportunity to refresh myself on some object-oriented programming and make something neat at the same time.

Here's the simulation running through 3 consecutive orders

Constructed a Python simulation of Amazon’s automated order fulfillment system using Object-Oriented Design concepts. Generated an event-based simulation in order to visualize and present the final design.
